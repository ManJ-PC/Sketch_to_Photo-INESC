Deep learning for face forensics é um sistema que usa um sketch de uma face (feito por exemplo por um perito de uma força policial) para gerar uma imagem semelhante a uma fotografia. Esta fotografia gerada pode então ser usada para procurar a pessoa que lhe corresponde numa base de dados de fotografias (por exemplo de "mugshots" de suspeitos).

Deep learning for face forensics is a system that uses a face sketch (done by a police expert for example) to generate an image that resembles a photo. This photo may then be used to search for the matching person in a database of photos (for example, a DB of mugshots of suspects).
